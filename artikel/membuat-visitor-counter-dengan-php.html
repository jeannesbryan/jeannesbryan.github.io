Membuat Visitor Counter dengan PHP

25 November 2022

Buat file baru lalu copy script di bawah ini. 

<?php
    $hitung = ("statistik.txt");
    $nilai = file($hitung);
    $nilai[0] ++;
    $fp = fopen($hitung, "w");
    fputs($fp, "$nilai[0]");
    fclose($fp);
?>
 
Dilihat <?php echo number_format($nilai[0], 0, '', '.'); ?> kali

Simpan file dengan ekstensi .php. Untuk menjalankan script diatas, pastikan Anda sudah menjalankan XAMPP (atau sejenisnya). Refresh browser untuk melihat hasilnya.
