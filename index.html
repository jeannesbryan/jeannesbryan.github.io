A personal terminal-style launcher for small offline-first web apps, experiments, and private browser tools.

> Install this launcher as a PWA for faster access.
> BUNKER_SOFTWARE
> Bunker is a software label for encrypted, offline-first browser tools stored under the /bunker directory.

Bunker Vault

Encrypted local vault for logins, private notes, payment cards, and Wi-Fi credentials. Import an .enc file, unlock it in memory, edit entries, then export a new encrypted file.

[ OPEN_VAULT ]

Bunker Auth

Encrypted local TOTP manager for second-factor secrets. Keep auth data separate from the password vault, generate codes locally, and export a protected .enc file.

[ OPEN_AUTH ]

Bunker Notes

Encrypted local Markdown notes for private writing, recovery plans, drafts, and personal documentation. Write in Markdown, preview safely in the browser, then export a protected .enc file.

[ OPEN_NOTES ]

Bunker Contacts

Encrypted local address book for private contacts, emergency numbers, work identities, and service records. Store contact data in memory only, then export a protected .enc file.

[ OPEN_CONTACTS ]

Bunker Ledger

Encrypted local ledger for personal transactions, flexible currencies, categories, accounts, and private financial notes. Track income and expenses locally, then export a protected .enc file.

[ OPEN_LEDGER ]

Bunker Board

Encrypted local Kanban board for private projects, tasks, priorities, due dates, tags, and offline-first workflows. Drag cards between columns, then export a protected .enc file.

[ OPEN_BOARD ]
